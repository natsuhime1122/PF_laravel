概要:

Laravelで製作した簡単なtweetアプリです。
使用した技術は最下部に記載しております。


目的:

Laravelの勉強のため製作いたしました。
まずCRUDの機能の実装を目標に製作を始め、その後バックエンドの知識を深めるためAjaxの勉強のため一部の機能をAjax通信で実装し直しました。

使用方法:

TOP画面のアカウント新規登録からユーザー登録、ログインからログインしていただけます。
ログイン画面下部にゲスト用アカウントのメールアドレスとパスワードを記載いたしましたので
コピペしていただけるとログインしていただけます。

ログイン後は投稿一覧画面にリダイレクトします。
投稿一覧画面上部の欄に文字を入力し送信ボタンを押すと、入力した内容を投稿できます。
投稿一覧画面の編集ボタンと削除ボタンはログインしているユーザーの投稿にしか表示されないようにしております。

ユーザーページからは現在ログインしているユーザーの投稿を一覧で見ることができます。
「ユーザー名を編集」をクリックするとユーザー名を変更できます。

投稿データのユーザー名をクリックすると、そのユーザーの投稿を一覧で見ることができます。
投稿データの本文をクリックすると、その投稿の詳細画面に遷移します。
詳細画面ではその投稿の全文とコメントを表示しております。
自分で投稿したコメントは編集・削除ができるようにしております。



仕様:

削除機能にはAjax通信を使用しており画面遷移なしに削除されるようにしております。
現在一部を除くデータはfactoryコマンドで挿入しております。
投稿詳細画面ではその投稿に対するコメントを入力できます。
コメント削除にはajax通信を使用し、削除時にコメント件数が更新されるようにしております。
