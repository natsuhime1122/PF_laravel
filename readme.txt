概要:

Laravelで製作した簡単なtweetアプリです。
使用した技術は最下部に記載しております。


目的:

Laravelの勉強のため製作いたしました。
まずCRUDの機能の実装を目標に製作を始めました。
その後バックエンドの知識を深めるためAjaxの勉強を始めました。
このアプリでは投稿データの削除にAjax通信を使用しております。


URL:

http://18.179.13.96/


ログイン方法:

ログイン画面下部にゲスト用アカウントのメールアドレスとパスワードを記載いたしましたので
コピペしていただけるとログインしていただけます。


使用方法:

TOP画面のヘッダーにある新規登録からユーザー登録、ログインからログインしていただけます。
ログイン後はヘッダーのLaravelTweetから投稿一覧画面に移動できます。
ユーザーページからは現在ログインしているユーザーの情報が閲覧できます。
新規投稿で新しい投稿データを送信でき、ログアウトからログアウトできます。

投稿一覧画面の編集ボタンと削除ボタンはログインしているユーザーの投稿にしか表示されないようにしております。
編集ボタンで投稿情報を編集できます。
削除ボタンで投稿情報を削除できます。
削除機能にはAjax通信を使用しており画面遷移なしに削除されるようにしております。






使用した技術
フロント:Bootstrap
言語:PHP Javascript jQuery(Ajax)
フレームワーク:Laravel
DB:MySQL
インフラ:AWS(EC2)
